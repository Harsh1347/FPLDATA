Team\_Data
=================

Allows you to access data related to Teams.

Use the below import to use Team_Data class

:code:`from fpldata import TeamData`

Create an instance of the class and then use the methods.

Example:

::
   
   t = TeamData()
   
   print(t.list_team_name())
